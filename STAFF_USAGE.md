# STAFF_USAGE

## 登录

在浏览器打开办公室局域网地址，输入共享访问码。

系统没有个人账号。上传报价单或生成报价单时，请在页面填写操作员姓名，方便经理查看操作记录。

## 上传完整报价单

1. 点击 `上传完整报价单`。
2. 填写操作员姓名。
3. 上传 `.xlsx` 报价文件。
4. 查看导入预览。
5. 如果预览显示 GTS、OEM、工厂或价格变化，请选择保留旧值或使用新值。
6. 如果预览显示单位变化，请确认提醒内容是否正确。
7. 如果匹配产品的非空产品字段不同，勾选需要更新的字段。
8. 如果缺少工厂、单位或价格，系统会禁止导入，请先修改 Excel。
9. 点击 `确认导入`。

系统会保留旧报价记录，并新增新的历史报价记录。若本次上传的 GTS/OEM、工厂、单位、价格与系统中已有报价完全一致，系统不会新增重复报价行，而是更新已有报价行的确认日期和更新人。MVP 不提供删除功能。

## 生成报价单

1. 点击 `生成报价单` 或首页的 `按 GTS/OEM 生成报价单`。
2. 填写操作员姓名。
3. 上传 `.xlsx` 需求清单。
4. 需求清单可以只包含 `GTS`，也可以包含 `GTS`、`OEM`、英文描述、数量、单位、备注等列。
5. 查看生成预览中的匹配、未匹配、冲突和多候选行。
6. 如果一行有多个候选报价，选择要使用的历史报价。
7. 如果不想使用某一行，取消勾选 `使用此行`。
8. 点击 `生成 Excel`。

如果需求文件中所有行都没有 GTS 或 OEM，系统会拒绝生成。若只有部分行缺少 GTS/OEM，系统会保留这些行并显示提醒。

生成文件只用于立即下载，MVP 不会永久保存到 `generated/`。

生成 Excel 时，如果上传的需求清单提供了 OEM、英文描述或单位，系统会保留上传值；如果这些字段为空，系统会使用选中的历史报价数据补齐。

## 查询数据库

点击 `查询数据库`，可按以下字段查询：

- GTS
- OEM
- 英文描述
- 品名
- 工厂

GTS/OEM 查询支持模糊匹配。例如搜索 `test001` 或 `001`，可以匹配到 `GTSTEST001`。如果存在完全匹配结果，系统只显示完全匹配结果。

查询操作目前不写入操作记录。

## HS Code

### 上传 HS Code

1. 点击 `HS Code` 或首页 `上传 HS Code`。
2. 填写操作员姓名。
3. 上传 `.xlsx` 文件。
4. 文件中必须有 GTS 列和 HS Code 列。
5. GTS 列使用系统现有 GTS 表头别名；HS Code 列支持 `HS`、`hscode`、`海关编码`、`hs code`。
6. 查看预览后点击 `确认更新`。

如果数据库里同一个 GTS 已经有 HS Code，系统会直接覆盖为本次上传的新 HS Code。系统只在操作记录里记录这次更新，不单独记录 HS Code 更新人或更新时间。

### 生成 HS Code 报表

1. 点击 `生成 HS Code 报表`。
2. 填写操作员姓名。
3. 上传包含 GTS 列的 `.xlsx` 文件。
4. 查看预览。
5. 点击 `下载 Excel`。

导出的 Excel 会保持上传文件中的 GTS 顺序，只包含 `GTS`、`OEM`、`HS Code` 三列。

## 操作记录

经理可以打开 `操作记录` 查看上传报价单、生成报价单、更新 HS Code、生成 HS Code 报表的记录。
